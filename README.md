# Prodesk-IT-mission-2
This is a simple Expense Tracker Dashboard where users can enter their salary, add expenses, and see the remaining balance updated instantly. The app helps users track their spending and manage their money easily with a clean and user-friendly interface.


## Features
Add total salary
Add expenses (name & amount)
View expense list
Real-time calculation of total expenses
Shows remaining balance instantly
Pie chart for visual summary (Balance vs Expenses)
Clear all data option


## Technologies Used
HTML
Tailwind CSS
JavaScript
Chart.js
